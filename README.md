# Store Next App

E-commerce Next App, bootstrapped by react | You can look and manage the products with a dashboard, calling data from the API.


## Features

- Offline support
- Add to homeScreen.
- Delete products
- Add products
- Edit products


## Dependencies

-Chart.js


## Scripts

* `npm install`
* `npm run dev`
